Docker for apache2 & PHP5 on latest Ubuntu
==============

Requirements
--------------

- docker: version 1.5

Installation
--------------

- Clone project
- Launch build: "docker build -t yougos666/docker-apache ."

Launch Docker container
--------------
docker run -d -p 80:80 yougos666/docker-apache


This docker  is available for:
- eZPublish 5.x
- Prestashop  >1.5.x
